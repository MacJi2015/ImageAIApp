## 查询官方模版列表（Effects TAB）

**接口地址**:`/facial/app/template/official`

**响应状态**:

| 状态码 | 说明         | schema           |
| ------ | ------------ | ---------------- |
| 200    | OK           | AppVideoTemplate |
| 401    | Unauthorized |                  |
| 403    | Forbidden    |                  |
| 404    | Not Found    |                  |

**响应参数**:
| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- |
|coverImageUrl|封面图片 URL|string||
|createdTime|创建时间|string(date-time)|string(date-time)|
|id|主键 ID|integer(int64)|integer(int64)|
|isOfficial|是否官方模版：1-是, 0-否|boolean||
|modifiedTime|修改时间|string(date-time)|string(date-time)|
|previewVideoUrl|预览视频 URL|string||
|promptText|AI 生成提示语|string||
|sortOrder|排序顺序|integer(int32)|integer(int32)|
|status|状态：ACTIVE-启用, INACTIVE-停用|string||
|templateId|模版 ID|string||
|templateName|模版名称|string||
|templateType|模版类型（动作类型）|string||

**响应示例**:

```javascript
[
  {
    coverImageUrl: '',
    createdTime: '',
    id: 0,
    isOfficial: true,
    modifiedTime: '',
    previewVideoUrl: '',
    promptText: '',
    sortOrder: 0,
    status: '',
    templateId: '',
    templateName: '',
    templateType: '',
  },
];
```

| 参数名称     | 参数说明                                       | 类型              | schema            |

| userId       | 用户 ID                                        | integer(int64)    | integer(int64)    |
