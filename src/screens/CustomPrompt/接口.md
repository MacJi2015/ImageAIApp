## 提交视频生成任务

**接口地址**:`/facial/app/video/generate`

**请求方式**:`POST`

**请求数据类型**:`application/json`

**响应数据类型**:`*/*`

**接口描述**:

**请求示例**:

```javascript
{
  "actionType": "",
  "duration": 5,
  "petImageUrl": "",
  "promptText": "",
  "removeWatermark": false,
  "shareToCommunity": false,
  "templateId": ""
}
```

**请求参数**:

**请求参数**:

| 参数名称                     | 参数说明                     | 请求类型 | 是否必须 | 数据类型             | schema               |
| ---------------------------- | ---------------------------- | -------- | -------- | -------------------- | -------------------- |
| token                        | 鉴权 token                   | header   | true     | string               |                      |
| request                      | 视频生成请求                 | body     | true     | VideoGenerateRequest | VideoGenerateRequest |
| &emsp;&emsp;actionType       | 动作类型                     |          | true     | string               |                      |
| &emsp;&emsp;duration         | 视频时长（秒）               |          | false    | integer(int32)       |                      |
| &emsp;&emsp;petImageUrl      | 宠物图片 URL                 |          | true     | string               |                      |
| &emsp;&emsp;promptText       | 用户输入的 AI 提示语（可选） |          | false    | string               |                      |
| &emsp;&emsp;removeWatermark  | 是否移除水印                 |          | false    | boolean              |                      |
| &emsp;&emsp;shareToCommunity | 是否分享到社区               |          | false    | boolean              |                      |
| &emsp;&emsp;templateId       | 使用的模版 ID（可选）        |          | false    | string               |                      |

**响应状态**:

| 状态码 | 说明         | schema            |
| ------ | ------------ | ----------------- |
| 200    | OK           | VideoTaskResponse |
| 201    | Created      |                   |
| 401    | Unauthorized |                   |
| 403    | Forbidden    |                   |
| 404    | Not Found    |                   |

**响应参数**:

| 参数名称      | 参数说明           | 类型           | schema         |
| ------------- | ------------------ | -------------- | -------------- |
| estimatedTime | 预计生成时间（秒） | integer(int32) | integer(int32) |
| status        | 状态               | string         |                |
| taskId        | 任务 ID            | string         |                |

**响应示例**:

```javascript
{
	"estimatedTime": 0,
	"status": "",
	"taskId": ""
}
```

## 查询任务状态

**接口地址**:`/facial/app/video/task/{taskId}`

**请求方式**:`GET`

**请求数据类型**:`application/x-www-form-urlencoded`

**响应数据类型**:`*/*`

**接口描述**:

**请求参数**:

**请求参数**:

| 参数名称 | 参数说明   | 请求类型 | 是否必须 | 数据类型 | schema |
| -------- | ---------- | -------- | -------- | -------- | ------ |
| taskId   | 任务 ID    | path     | true     | string   |        |
| token    | 鉴权 token | header   | true     | string   |        |

**响应状态**:

| 状态码 | 说明         | schema       |
| ------ | ------------ | ------------ |
| 200    | OK           | AppVideoTask |
| 401    | Unauthorized |              |
| 403    | Forbidden    |              |
| 404    | Not Found    |              |

**响应参数**:

| 参数名称          | 参数说明                                                           | 类型              | schema            |
| ----------------- | ------------------------------------------------------------------ | ----------------- | ----------------- |
| actionType        | 动作类型                                                           | string            |                   |
| aiServiceProvider | AI 服务提供商                                                      | string            |                   |
| aiTaskId          | AI 服务任务 ID                                                     | string            |                   |
| createdTime       | 创建时间                                                           | string(date-time) | string(date-time) |
| duration          | 视频时长（秒）                                                     | integer(int32)    | integer(int32)    |
| errorMessage      | 错误信息                                                           | string            |                   |
| id                | 主键 ID                                                            | integer(int64)    | integer(int64)    |
| modifiedTime      | 修改时间                                                           | string(date-time) | string(date-time) |
| petImageUrl       | 宠物图片 URL                                                       | string            |                   |
| promptText        | 用户输入的 AI 提示语                                               | string            |                   |
| removeWatermark   | 是否移除水印：1-是, 0-否                                           | boolean           |                   |
| shareToCommunity  | 是否分享到社区：1-是, 0-否                                         | boolean           |                   |
| status            | 状态：PENDING-待处理, PROCESSING-生成中, SUCCESS-成功, FAILED-失败 | string            |                   |
| taskId            | 任务 ID（UUID）                                                    | string            |                   |
| templateId        | 使用的模版 ID                                                      | string            |                   |
| thumbnailUrl      | 缩略图 URL                                                         | string            |                   |
| userId            | 用户 ID                                                            | integer(int64)    | integer(int64)    |
| videoUrl          | 生成的视频 URL                                                     | string            |                   |

**响应示例**:

```javascript
{
	"actionType": "",
	"aiServiceProvider": "",
	"aiTaskId": "",
	"createdTime": "",
	"duration": 0,
	"errorMessage": "",
	"id": 0,
	"modifiedTime": "",
	"petImageUrl": "",
	"promptText": "",
	"removeWatermark": true,
	"shareToCommunity": true,
	"status": "",
	"taskId": "",
	"templateId": "",
	"thumbnailUrl": "",
	"userId": 0,
	"videoUrl": ""
}
```
